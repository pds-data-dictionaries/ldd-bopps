# Regression Test Directory

Because the BOPPS mission started and ended in 2014 and the data have been 
archived for over a decade, no further development is anticipated on this
dictionary. A valid test label is provided for sanity-checking, but not other 
test labels have been prepared.

Note that this is not a valid example label for BOPPS calibrated data. It has
been stripped of references to all schema files and to namespaces other
than bopps: and pds: core.
